AIPS常用工具

  1. uvplt 展示uv覆盖

    task ‘uvplt’

  2. possm 展示相位与幅度随频率的变化

    task ‘possm’

    ‘b’/‘c’翻页，’d’退出

  3. accor 归一化可见度

    task ‘accor’

  4. snplt 展示新SN表（可展示CL表或TY表）

  5. clcal 将SN表应用到CL表上

  6. vlbautil

    run vlbautil

    vlbamcal

  7. setjy 设置流量密度

    task ‘setjy’

  8. apcal 应用参数到SN表中，如SEFD

    task ‘apcal’

  9. fring 条纹拟合

    task ‘fring’

  10. bpass 带通校准，将带通校准的信息写入BP表

    task ‘bpass’

  11. tget 参数与上次运行的相同

  12. split 应用CL表并沿频率积分

AIPS校准的过程可以简单概括为：SN.(Version)的建立，SN表应用到CL表来更新CL.(Version)表

流程:

第一步校准：归一化可见度

accor$\rarr$ snplt(查看SN表)$\rarr$clcal$\rarr$snplt(查看CL表)

得到SN.1并应用于CL.1，得到CL.2

第二步校准：校正SEFD

vlbautil(得到GC和TY表)$\rarr$setjy$\rarr$apcal(应用SEFD于SN表中)$\rarr$snplt$\rarr$clcal

得到SN.2并应用于CL.2，得到CL.3

==至此，幅度校准完毕，即将进行相位校准。==

第三步校准：条纹拟合（时延校正）

fring(得到残余时延应用于SN表)$\rarr$snplt(查看SN表)$\rarr$clcal$\rarr$snplt(查看CL表)

得到SN.3并应用于CL.3，得到CL.4

第四步校准：带通校准(先幅度和后相位)

bpass(获得带通校准信息，写入BP表)$\rarr$possm(查看BP表)$\rarr$(tget) bpass(续写BP表)$\rarr$(tget) possm

<(tget)为带通校正的相位校正部分。>

得到BP表

第五步：查看校准结果

(tget) possm

应用CL.4表，得到校准后的可见度数据

==至此，相位校准结束。可见度校准结束==

第六步：应用最终的CL表

split

将第一到五步得到CL表应用于未校准可见度，输出校准后的可见度数据

第七步：输出fits文件

fittp

将校准后的可见度数据输出至fits文件中，用于后续difmap成图处理或者相位参考。

VLBA Pipeline，带有vlba开头的procedure都是专用于处理vlba产生的数据。

  1. run vlbautil

vlbautil是个run file，里头定义了VLBA数据处理的一些procedures，可以看作是脚本

  1. vlbaecr

    procedure，校正电离层时延

  2. vlbaeops

    procedure，校正地球自转影响

  3. vlbaccor

    procedure，校正由于数字采样带来的信号损失率

  4. vlbacrpl

    procedure，查看条纹，调用possm

  5. vlbampcl

    procedure，校正设备时延

  6. vlbabpss

    procedure，校正带通影响

  7. vlbaamp

    procedure，校正幅度

  8. vlbapang

    procedure，校正视差角
